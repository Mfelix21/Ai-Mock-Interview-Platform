# AI Mock Interview Platform

Full-stack interview preparation platform that delivers role-specific interview questions through a React frontend and FastAPI backend. The platform helps users practice technical, behavioral, project-based, and scenario-based interview questions while preparing for software engineering, data analyst, and data science roles.

---

## Overview

The goal of this project is to build an end-to-end interview preparation platform that demonstrates full-stack software engineering concepts including:

* Frontend development with React
* Backend development with FastAPI
* REST API design
* Database integration
* Object-Oriented Programming (OOP)
* Cloud deployment
* Docker containerization
* CI/CD automation
* AI-powered feedback and recommendations

---

## Features

- Role-specific interview questions
- User registration and login
- Secure password hashing (bcrypt)
- PostgreSQL database integration
- User-specific interview history
- Protected answer ownership
- Saved interview responses
- Responsive modern UIrn UI
